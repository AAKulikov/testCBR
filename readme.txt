#Описание проекта#
_cbr - папка для обновления курсов
	/_cbr/_update.php - можно поставить на cron можно обратиться в ручном режиме 

_data - папка со скриптами
	/mpm.php - сборка страниц
	/_kit_engine.php - все обработки
	
	/_modules/
		/_currency/_list.php - скрипт вывода данных по валютам
		
_lib/ - функции и прочие технические файлы, в том числе подключение к Mysql (фаил) _sql.php

storage/ - папка в которую сохраняются запросы на получение json с графиков

tpl/ - шаблон
